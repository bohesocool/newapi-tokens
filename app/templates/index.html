    new-api 渠道启停控制
    填入 new-api 管理员的「访问令牌 (access_token)」与「用户 ID」，即可在仪表盘卡片上直接启用 / 暂停渠道。三者均必填。
    获取方式：登录 new-api → 个人设置 → 生成「访问令牌」；用户 ID 在用户管理页查看。令牌不回显，留空表示不修改。
    API Key (外部调用鉴权)
    将此 Key 放入请求头即可调用所有 /api/* 数据接口。请妥善保管。
    重置后旧 Key 立即失效，所有使用旧 Key 的外部调用都需要更新。

    Webhook 推送
    报表生成后由后端定时器自动 POST 到此 URL，通用 JSON：{type, text, data, timestamp}，多数接收端可直接用 text 字段。
    触发时机：每个整点存完快照后推送上一小时的小时报；每日 0 点归档后推送前一日日报。
    错误率告警：每分钟统计各渠道 失败/(成功+失败)，该分钟总调用 ≥20 且错误率 ≥10% 即推送；恢复正常补发一条「已恢复」。
